# hourglass
A fast and easy time tracking CLI

## Usage

```
> hourglass help
A fast, easy-to-use and lightweight time tracking tool, written in Go.

Usage:
  hourglass [command]

Available Commands:
  delete      Delete a log
  help        Help about any command
  log         Log time 
  summary     Sum up logs 

Flags:
      --config string   config file (default is $HOME/.config/hourglass/config)
  -h, --help            help for hourglass

Use "hourglass [command] --help" for more information about a command.
```
